# Labb 2

This repository contains my work for assignment 2 in machine learning.

## Files

- `labb2.ipynb`: notebook for the lab
- `imagenet_class_index.json`: ImageNet class labels
- `images/`: test images used in the notebook

## Setup

Create a virtual environment and install the dependencies:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Run

Open `labb2.ipynb` in VS Code or Jupyter and run the cells in order.
